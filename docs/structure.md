# Project structure

* `crawler/`
  Contains a crawler that gets data from the web, to fill our database with products.
* `docs/`  
  Documentation.
* `static/`
  Static files like image files, will be saved here.
* `public/`
  * `css/`
      For the css files.
* `routes/`
  Basic routes for the project.
* `views/`
  Views (html files) saved here.
* `app.js`  
  Main app.
